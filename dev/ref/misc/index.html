Miscellaneous

SymPh.State — Type
State(mesh)

Initializes a State object that holds a list of arrays corresponding to the characteristics of mesh. Whenever trying to access a field, the state checks if the object is in its fields, in which case it returns it, and otherwise it allocates it and then returns it.
